# MiniFE

MiniFE is a simple frontend project built with **React** and **Vite**, using **Redux Toolkit** for state management and **Axios** for API communication. The project is designed as a lightweight starting point for learning or building small to medium-scale frontend applications.

## üì¶ Dependencies Explanation

### Dependencies

- **react / react-dom**: Core libraries for building and rendering the UI
- **redux**: Predictable state container for JavaScript apps
- **@reduxjs/toolkit**: Simplifies Redux logic (slices, reducers, actions)
- **react-redux**: Connects Redux store with React components
- **axios**: Handles HTTP requests to backend APIs
- **bootstrap**: Provides ready-to-use UI components and responsive design

> ‚ö†Ô∏è Note: `boostrap` is listed but not required. You should remove it and keep only `bootstrap`.

### Dev Dependencies

- **vite**: Development server and build tool
- **@vitejs/plugin-react**: Enables React support in Vite
- **eslint**: Linting tool to maintain clean code
- **eslint-plugin-react-hooks**: Enforces best practices for React Hooks
- **eslint-plugin-react-refresh**: Improves hot reload experience
- **@types/react / @types/react-dom**: Type definitions for React (useful if migrating to TypeScript)
